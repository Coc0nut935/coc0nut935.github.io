# 力扣刷题-[239. 滑动窗口最大值](https://leetcode.cn/problems/sliding-window-maximum/)

## 题目：

给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 `k` 个数字。滑动窗口每次只向右移动一位。

返回 *滑动窗口中的最大值* 。

**示例 1：**

```
输入：nums = [1,3,-1,-3,5,3,6,7], k = 3
输出：[3,3,5,5,6,7]
解释：
滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```

## 题解：

### 暴力枚举：

我第一次做的时候是直接用暴力做的，需要遍历数组中所有可能的滑动窗口起始位置，对于每一个起始位置，我们都单独扫描该窗口内的全部 `k` 个元素，通过比较找出其中的最大值并记录下来。

### 单调队列：

需要维护一个队列，其中存储数组元素的**索引**，并保证队列中索引对应的数值严格递减（即队首始终是当前窗口的最大值）。遍历数组时，每加入一个新元素，就先将队尾所有小于等于该元素的索引移除（因为它们在此元素出现后不再可能成为最大值），然后将新索引入队。之后检查队首索引是否已滑出窗口，若已滑出则将其移除。当窗口大小达到 k 时，队首索引对应的元素即为当前窗口最大值，直接记录到结果数组中。

## 代码：

```Java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        /**方法一：暴力枚举
        int n = nums.length;
        if (n == 0 || k == 0) return new int[0];
        
        int[] ans = new int[n - k + 1];
        
        // 枚举每个窗口的起始位置 left
        for (int left = 0; left <= n - k; left++) {
            // 初始化当前窗口最大值为极小值
            int max = Integer.MIN_VALUE;
            // 遍历窗口内的所有元素 [left, left + k - 1]
            for (int right = left; right < left + k; right++) {
                if (nums[right] > max) {
                    max = nums[right];
                }
            }
            ans[left] = max;
        } */
        //方法二：单调队列
        int n = nums.length;
        int[] ans = new int[n - k + 1];
        Deque<Integer> deque = new ArrayDeque<>();
        for(int i = 0;i<n;i++){
            while(!deque.isEmpty()&&nums[deque.peekLast()] <= nums[i]){
                deque.pollLast();
            }
            //将当前索引加入队尾
            deque.offerLast(i);
            //删除已经不再窗口内的索引
            if(deque.peekFirst()<=i - k){
                deque.pollFirst();
            }
            if(i>=k - 1){
                ans[i - k + 1] = nums[deque.peekFirst()];
            }
        }
        return ans;
    }
}
```
