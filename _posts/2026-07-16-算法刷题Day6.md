# 力扣刷题-[42. 接雨水](https://leetcode.cn/problems/trapping-rain-water/)

## 题目：

给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

**示例 1：**

![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2018/10/22/rainwatertrap.png)

```
输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 
```

## 题解：

### 前后缀分解：

这个问题的关键在于要知道**每个柱子能容纳的水量取决于其左右两侧最高柱子中较矮的那一个**。可以使用“前后缀分解”的方法：预先从左到右扫描数组，记录每个位置左侧（含自身）的最高柱子高度（前缀最大值）；再从右到左扫描，记录每个位置右侧（含自身）的最高柱子高度（后缀最大值）。对于每个柱子，其上方能接的水量等于 `min(左侧最高, 右侧最高) - 自身高度`，将这个差值累加即为总接水量。

### 双指针：

我们让左右指针从两端向中间靠拢，并实时维护两侧扫描过的最高柱子高度。**哪一侧的当前柱子更矮，就先结算哪一侧的雨水量**。因为对于较矮的一侧，其外侧已经有一根更高的柱子作为挡板，它的蓄水上限必然受限于同侧已经扫描过的最高值，与另一侧尚未扫描的部分无关。这种“以矮定局”的贪心策略，使得我们能够一边移动指针一边累加水量，最终在一次遍历中精准计算出总接水量。

## 代码：

```java
class Solution {
    public int trap(int[] height) {
        /*方法一：前后缀分解
        int n = height.length;
        int[] premax = new int[n];
        premax[0] = height[0];
        //计算前缀最大值
        for(int i = 1;i<n;i++){
            premax[i] = Math.max(premax[i-1],height[i]); 
        }
        int[] surmax = new int[n];
        surmax[n-1] = height[n -1];
        //计算后缀最大值
        for(int i = n-2;i>=0;i--){
            surmax[i] = Math.max(surmax[i+1],height[i]);
        }
        //计算每个单位能够接多少水
        int ans = 0;
        for(int i=0;i<n;i++){
            ans = ans + Math.min(premax[i],surmax[i]) - height[i];
        }*/
        //方法二：双指针
        int n = height.length;
        int premax = 0;
        int surmax = 0;
        int left = 0;
        int right = n - 1;
        int ans = 0;
        while(left<=right){
            premax = Math.max(premax,height[left]);
            surmax = Math.max(surmax,height[right]);
            if(premax<surmax){
                ans = ans + premax - height[left];
                left++;
            }
            else{
                ans = ans + surmax - height[right];
                right--;
            }
        }
        return ans;
    }
}
```
