# 力扣刷题-[141. 环形链表](https://leetcode.cn/problems/linked-list-cycle/)

## 题目：

给你一个链表的头节点 `head` ，判断链表中是否有环。

如果链表中有某个节点，可以通过连续跟踪 `next` 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 `pos` 来表示链表尾连接到链表中的位置（索引从 0 开始）。**注意：`pos` 不作为参数进行传递** 。仅仅是为了标识链表的实际情况。

*如果链表中存在环* ，则返回 `true` 。 否则，返回 `false` 。

**示例 1：**

![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2018/12/07/circularlinkedlist.png)

```
输入：head = [3,2,0,-4], pos = 1
输出：true
解释：链表中有一个环，其尾部连接到第二个节点。
```

## 题解：

这道题还是使用昨天学过的 **快慢指针**。我们初始化两个指针 `slow` 和 `fast`，都指向链表头节点。在每次迭代中，`slow` 向前移动一步，`fast` 向前移动两步。如果链表中存在环，那么 `fast` 最终会追上 `slow`（两者相遇），就像在环形跑道上速度快的选手会追上慢的选手一样。如果链表无环，`fast` 会首先到达链表末尾（`fast == null` 或 `fast.next == null`），此时循环结束，返回 `false`。

## 代码：

```java
public class Solution {
    public boolean hasCycle(ListNode head) {
        //这一题是使用快慢指针做，当两个指针相遇了，那就说明这个链表是一个环形链表
        ListNode slow = head;
        ListNode fast = head;
        while(fast != null&&fast.next!=null){
            slow = slow.next;
            fast = fast.next.next;
            if(fast == slow){
                return true;
            }
        }
        return false;
    }
}
```

# 力扣刷题-[142. 环形链表 II](https://leetcode.cn/problems/linked-list-cycle-ii/)

## 题目：

给定一个链表的头节点  `head` ，返回链表开始入环的第一个节点。 *如果链表无环，则返回 `null`。*

如果链表中有某个节点，可以通过连续跟踪 `next` 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 `pos` 来表示链表尾连接到链表中的位置（**索引从 0 开始**）。如果 `pos` 是 `-1`，则在该链表中没有环。**注意：`pos` 不作为参数进行传递**，仅仅是为了标识链表的实际情况。

**不允许修改** 链表。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2018/12/07/circularlinkedlist.png)

```
输入：head = [3,2,0,-4], pos = 1
输出：返回索引为 1 的链表节点
解释：链表中有一个环，其尾部连接到第二个节点。
```

## 题解：

这道题依然是使用 **快慢指针**，但还额外增加了数学推导。我们首先让快指针（每次两步）和慢指针（每次一步）遍历链表。若两者相遇，则证明链表中存在环，且相遇点一定在环内。接下来，将快指针重新指向链表头部，然后让两个指针以**相同的速度（每次一步）**同步前进。根据严格的数学推导（`a = (n-1)L + c`），它们第二次相遇的位置必然是环的入口节点。（**这个数学推理过程是看0x3f大佬的**）

## 代码：

```java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        ListNode slow = head;
        ListNode fast = head;
        while(fast != null&&fast.next != null){
            slow = slow.next;
            fast = fast.next.next;
            //第一次相遇
            if(fast == slow){
                fast = head;
                while(fast != slow){
                    fast = fast.next;
                    slow = slow.next;
                }
                return slow;
            }
        }
        return null;
    }
}
```

# 力扣刷题-[143. 重排链表](https://leetcode.cn/problems/reorder-list/)

## 题目：

给定一个单链表 `L` 的头节点 `head` ，单链表 `L` 表示为：

```
L0 → L1 → … → Ln - 1 → Ln
```

请将其重新排列后变为：

```
L0 → Ln → L1 → Ln - 1 → L2 → Ln - 2 → …
```

不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

**示例 1：**

![img](https://pic.leetcode.cn/1626420311-PkUiGI-image.png)

```
输入：head = [1,2,3,4]
输出：[1,4,2,3]
```

## 题解：

分为三步 **“找中点 + 反转后半部分 + 合并链表”** 。首先利用快慢指针定位链表的中间节点，然后将后半部分链表原地反转。此时，链表被拆分为两个独立的子链表：前半部分保持原序，后半部分逆序。最后，我们同时遍历这两个子链表，交替地将后半部分的节点插入到前半部分的节点之间，形成 `L0 → Ln → L1 → Ln-1 → ...` 的新顺序。

## 代码：

```java
class Solution {
    public void reorderList(ListNode head) {
        //这个逻辑跟反转链表是差不多的
        ListNode slow = head;
        ListNode fast = head;
        while(fast != null&&fast.next != null){
            slow = slow.next;
            fast = fast.next.next;
        }

        //对后半部分的链表进行反转
        ListNode preNode = null;
        ListNode curnode = slow;
        while(curnode != null){
            ListNode tempnext = curnode.next;
            curnode.next = preNode;
            preNode = curnode;
            curnode = tempnext;
        }

        //合并链表
        ListNode head2 = preNode;
        while(head2.next != null){
            ListNode tempnext = head.next;
            ListNode tempnext2 = head2.next;
            head.next = head2;
            head2.next = tempnext;
            head = tempnext;
            head2 = tempnext2; 
        }
    }
}
```
