# 力扣刷题-[160. 相交链表](https://leetcode.cn/problems/intersection-of-two-linked-lists/)

## 题目：

给你两个单链表的头节点 `headA` 和 `headB` ，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 `null` 。

图示两个链表在节点 `c1` 开始相交**：**

[![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2018/12/14/160_statement.png)](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2018/12/14/160_statement.png)

题目数据 **保证** 整个链式结构中不存在环。

## 题解：

解出这道题可以使用**双指针同步遍历法**。可以让指针 `p` 从链表 `headA` 出发，指针 `q` 从 `headB` 出发，同步向后移动。当 `p` 到达 `headA` 的末尾时，将其重定向到 `headB` 继续前进；同样，当 `q` 到达 `headB` 的末尾时，将其重定向到 `headA` 继续前进。这样一来，两个指针各自走过的总路程都会变成 `lenA + lenB`（两个链表的长度之和）。如果两个链表存在交点，那么它们必然会在走过相同长度的路程后，同时到达那个交点——因为从交点往后到末尾的公共部分是共享的，消除长度差后，剩余的差值恰好被重定向前的路径弥补。如果不存在交点，它们最终会同时变成 `null`，并结束循环。

## 代码：

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode p = headA;
        ListNode q = headB;
        while(p!=q){
            p = p != null ? p.next:headB;
            q = q != null ? q.next:headA;
        }
        return p;
    }
}
```



# 力扣刷题-[206. 反转链表](https://leetcode.cn/problems/reverse-linked-list/)

## 题目：

给你单链表的头节点 `head` ，请你反转链表，并返回反转后的链表。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2021/02/19/rev1ex1.jpg)

```
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]
```

## 题解：

我使用的是**头插法**来解决这道题，是通过遍历原链表，将每个节点依次从旧链表中取出然后插入到新链表的头部，从而完成反转。

## 代码：

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        //头插法
        ListNode prenode = null;
        ListNode curnode = head;
        while(curnode != null){
            ListNode nextnode = curnode.next;
            curnode.next = prenode;
            prenode = curnode;
            curnode = nextnode;
        }
        return prenode;
    }
}
```

# 力扣算题-[876. 链表的中间结点](https://leetcode.cn/problems/middle-of-the-linked-list/)

## 题目：

给你单链表的头结点 `head` ，请你找出并返回链表的中间结点。

如果有两个中间结点，则返回第二个中间结点。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2021/07/23/lc-midlist1.jpg)

```
输入：head = [1,2,3,4,5]
输出：[3,4,5]
解释：链表只有一个中间结点，值为 3 。
```

## 题解：

使用的是 **快慢指针法**。我们让一个慢指针 `slow` 每次走一步，快指针 `fast` 每次走两步。当快指针到达链表末尾时，慢指针恰好会停在链表的中间位置。具体来说，如果链表长度为奇数（如 `1->2->3->4->5`），快指针会走到最后一个结点 `5`，此时慢指针在 `3`，即正中间；如果链表长度为偶数（如 `1->2->3->4`），快指针会走到 `null`（越过最后一个结点），此时慢指针在 `3`，即第二个中间结点（题目要求返回第二个）。

## 代码：

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode middleNode(ListNode head) {
        //这一题就是对快慢指针有一个很好的理解
        ///链表的长度为奇数的时候，快指针指向尾节点的时候，慢指针就指向中间
        //指针的长度为偶数的时候，快指针的下一个节点为空的时候，慢指针就指向中间
        ListNode slow = head;
        ListNode fast = head;
        while(fast != null && fast.next !=null){
            slow = slow.next;
            fast = fast.next.next;
        }
        return slow;
    }
}
```





题目：

给你一个单链表的头节点 `head` ，请你判断该链表是否为回文链表。如果是，返回 `true` ；否则，返回 `false` 。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2021/03/03/pal1linked-list.jpg)

```
输入：head = [1,2,2,1]
输出：true
```

题解：

这道题就是使用了前面所学的**头插法**和**快慢指针**，代码跟前面的也是完全一样的，有一点不一样的就是，在进行链表反转的时候，是从中间节点开始往后面反转（也就是反转后半部分），所以在反转的时候其实节点为慢指针所对应的节点

代码：

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public boolean isPalindrome(ListNode head) {
        //这一题可以使用快慢指针和反转链表进行解题
        //1.快慢指针找中点
        ListNode slow = head;
        ListNode fast = head;
        while(fast != null && fast.next != null){
            slow = slow.next;
            fast = fast.next.next;
        }

        //反转后半部分的链表
        ListNode prenode = null;
        ListNode curnode = slow;
        while(curnode!=null){
            ListNode nextnode = curnode.next;
            curnode.next = prenode;
            prenode = curnode;
            curnode = nextnode;
        }
        
        //3.比较前半部分和反转后的后半部分
        ListNode p1 = head;
        ListNode p2 = prenode;
        while (p2 != null) {
            if (p1.val != p2.val) return false;
            p1 = p1.next;
            p2 = p2.next;
        }
        return true;
    }
}
```
