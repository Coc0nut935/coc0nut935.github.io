今天主要学习了滑动窗口的题目

**滑动窗口**本质上是由左右两个指针在数组上划定的一段连续区间，随着指针的移动，这个区间像窗口一样在数组上平滑滑动。

# 力扣刷题-长度最小的子数组

[209. 长度最小的子数组](https://leetcode.cn/problems/minimum-size-subarray-sum/)

## 题目：

给定一个含有 `n` 个正整数的数组和一个正整数 `target` **。**

找出该数组中满足其总和大于等于 `target` 的长度最小的 **子数组** `[numsl, numsl+1, ..., numsr-1, numsr]` ，并返回其长度**。**如果不存在符合条件的子数组，返回 `0` 。

## 解题：

### 暴力：

用外层循环固定子数组的起始位置 `i`，内层循环从 `i` 开始向后累加元素，一旦当前累加和 `sum` 大于等于 `target`，就计算当前子数组长度 `j - i + 1` 并更新答案，然后跳出内层循环（因为继续向后累加长度只会更长，不可能更优）。若遍历完所有起始位置仍未找到满足条件的子数组，则返回 0。

### 滑动窗口：

我们维护一个动态窗口，用右指针不断向右扩展以纳入新元素，并实时累加窗口内元素和。一旦窗口和大于等于目标值 `target`，说明当前窗口满足条件，此时记录窗口长度并尝试收缩左指针，同时从和中减去移出的元素，以寻找更短的可行窗口。反复执行此过程，直到右指针遍历完整个数组。在整个过程中，我们始终保持窗口是连续的，并且通过“满足条件时收缩左边界”的策略，这种方法只需遍历数组一次。

## 代码：

```java
class Solution {
    public int minSubArrayLen(int target, int[] nums) {
        /*方法一：暴力枚举(会超时)
        int n = nums.length;
        int ans = n+1;
        for(int i = 0;i<n;i++){
            int sum = 0;
            for(int j = i;j<n;j++){
                sum = sum+nums[j];
                if(sum>=target){
                    ans = Math.min(ans,j-i+1);
                    //结束当前内循环
                    break;
                }
            }
        }
        return ans == n+1 ? 0 : ans;*/
        //方法二：双指针（滑动窗口）
        int n = nums.length;
        int ans = n+1;
        int left = 0;
        int sum = 0;
        for(int right = 0;right<n;right++){
            //向右扩增元素
            sum +=nums[right];
            //将左窗口向右收缩
            while(sum>=target){
                ans = Math.min(ans,right - left + 1);
                //将当前左下标的元素移出sum
                sum -= nums[left];
                left++;
            }
        }
        return ans == n+1 ? 0 : ans;
    }
}
```

# 力扣刷题-[ 乘积小于 K 的子数组](https://leetcode.cn/problems/subarray-product-less-than-k/)

## 题目：

给你一个整数数组 `nums` 和一个整数 `k` ，请你返回子数组内所有元素的乘积严格小于 `k` 的连续子数组的数目。

**示例 1：**

```
输入：nums = [10,5,2,6], k = 100
输出：8
解释：8 个乘积小于 100 的子数组分别为：[10]、[5]、[2]、[6]、[10,5]、[5,2]、[2,6]、[5,2,6]。
需要注意的是 [10,5,2] 并不是乘积小于 100 的子数组。
```

## 解题：

这道题还是使用滑动窗口，通过右指针不断向右扩展，将新元素乘入窗口乘积。一旦窗口乘积大于等于 `k`，就持续将左指针右移并从乘积中移除对应元素，直到乘积重新小于 `k`。此时，**以当前右指针为结尾的所有连续子数组**（从 `left` 到 `right` 的任意起点开始）的乘积必然都小于 `k`，其个数正好等于 `right - left + 1`。将这些个数累加到答案中，即可得到所有满足条件的子数组总数。

## 代码：

```java
class Solution {
    public int numSubarrayProductLessThanK(int[] nums, int k) {
        //如果k为0,那么就不会有连续的子数组满足
        if(k==0){
            return 0;
        }
        int n = nums.length;
        int prod = 1;
        int ans = 0;
        int left = 0;
        for(int right = 0;right < n ;right++){
            prod = prod*nums[right];
            while(prod>=k){
                //当乘积大于等于k的时候就需要对左窗口进行收缩
                prod = prod/nums[left];
                left++;
            }
            ans = ans + right - left + 1;
        }
        return ans;
    }
}
```

# 力扣刷题-[ 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

## 题目：

给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长 子串** 的长度。

**示例 1:**

```
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。
```

## 题解：

这道题还是使用滑动窗口，但会通过哈希表记录窗口内每个字符最近一次出现的位置。我们让右指针 `right` 不断向右扩展，将新字符纳入窗口，同时通过哈希表检查该字符是否已经在当前窗口中出现过。若出现过，则将左指针 `left` 直接移动到该字符上一次出现位置的下一个索引，从而确保窗口内始终不包含重复字符。每次调整完窗口后，当前窗口长度 `right - left + 1` 即为以 `right` 结尾的最长无重复子串长度，用它更新全局最大值。在整个过程中，每个字符最多被访问两次（右指针进入一次，左指针跳过一次）。

## 代码：

```Java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        int n = s.length();
        int ans = 0;
        int left = 0;
        //用于记录字符近一次出现的下标
        Map<Character, Integer> lastIndex = new HashMap<>();
        for(int right = 0;right<n;right++){
            char c = s.charAt(right);
            //如果该字符已出现过，并且其位置在窗口内，则左窗口跳到该字符最后一次出现下标的下一个位置
            if(lastIndex.containsKey(c)&&lastIndex.get(c)>=left){
                left = lastIndex.get(c) + 1;
            }
            //更新当前字符最近一次遇到的下标
            lastIndex.put(c, right);
            ans = Math.max(ans,right - left + 1);
        }
        return ans;
    }
}
```
