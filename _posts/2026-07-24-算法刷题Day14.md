# 力扣刷题-[19. 删除链表的倒数第 N 个结点](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/)

## 题目：

给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/10/03/remove_ex1.jpg)

```
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]
```

## 题解：

我使用的是 **“两次遍历，定位删除”**。首先，我们遍历整个链表，计算出链表的总长度 `cnt`。这样，倒数第 `n` 个节点在正数方向上的位置就是 `cnt - n`（从 0 开始计数）。接着，我们再次从头遍历链表，找到该位置节点的前驱节点。如果该位置是 0（即要删除头节点），则直接让 `head` 指向下一个节点；否则，将前驱节点的 `next` 指针直接指向要删除节点的下一个节点，从而完成删除操作。

## 代码：

```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        int cnt = 0;
        ListNode curnode = head;
        while(curnode != null){
            curnode = curnode.next;
            cnt++;
        }
        if (cnt - n == 0) {
            return head.next;
        }
        ListNode pre = head;
        for(int i = 0;i<cnt - n - 1;i++){
            pre = pre.next;
        }
        pre.next = pre.next.next; 
        return head;
    }
}
```
