# 力扣Hot100-盛最多水的容器

[盛最多水的容器](https://leetcode.cn/problems/container-with-most-water/)

## 题目：

给定一个长度为 `n` 的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

**说明：**你不能倾斜容器。

## 题解：

可以使用“双指针”策略，其核心在于利用容器的容量由两条垂线中较短的那条决定这一物理特性。初始时，将左指针置于数组起始位置，右指针置于末尾，此时容器底边最长。通过计算当前左右指针围成的面积并更新全局最大值后，下一步的关键是移动指针——**移动高度较小的那一端**。因为面积取决于短板高度，若移动较高的指针，底边虽缩短，但高度只会受限于更小的那个值，容量不可能增加；反之，移动较矮的指针，则有机会遇到更高的垂线，从而在底边缩小的同时提升高度，获得更大的面积。如此反复，每次排除一个不可能成为最优解的边界，直至左右指针相遇。整个过程只需遍历一次数组，时间复杂度为 O(n)，空间复杂度为 O(1)，既高效又简洁地解决了这一经典最优化问题。

## 代码：

```java
class Solution {
    public int maxArea(int[] height) {
        int n = height.length;
        int left = 0;
        int right = n - 1;
        int ans = 0;
        while(left<right){
            //计算面积
            int area = (right - left)*Math.min(height[left],height[right]);
            //比较哪个面积大
            ans = Math.max(ans,area);
            if(height[left]<height[right]){
                left++;
            }
            else{
                right--;
            }
        }
        return ans;
    }
}
```
