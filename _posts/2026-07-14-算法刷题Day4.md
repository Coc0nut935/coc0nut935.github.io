# 力扣刷题-[438. 找到字符串中所有字母异位词](https://leetcode.cn/problems/find-all-anagrams-in-a-string/)

## 题目：

给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的 **异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

**示例 1:**

```
输入: s = "cbaebabacd", p = "abc"
输出: [0,6]
解释:
起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。
```

## 题解：

这题的解法是 **固定长度的滑动窗口 + 字符计数数组**。由于字符串仅包含小写字母，可以使用长度为 26 的数组 `cntP` 记录目标字符串 `p` 中每个字符的出现次数，再维护一个同样长度的数组 `windowCnt` 记录当前窗口中每个字符的出现次数。窗口大小固定为 `p.length()`，我们从左到右滑动窗口：每向右移动一位，就纳入一个新字符（增加其计数），并移除一个旧字符（减少其计数）。在每次窗口滑动后，比较两个计数数组是否完全相同。若相同，则说明当前窗口内的子串是 `p` 的一个异位词，记录其起始索引。

## 代码：

```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        //用集合记录起始位置
        List<Integer> ans = new ArrayList<>();
        int m = s.length();
        int n = p.length();
        //用数组记录字符出现的次数
        int[] cntP = new int[26];
        int[] windowCnt = new int[26];
        //记录字符串p中字符出现的次数
        for(char c : p.toCharArray()){
            cntP[c - 'a']++;
        }
        int left = 0;
        for(int right = 0;right<m;right++){
            windowCnt[s.charAt(right) - 'a']++;
            //当窗口的大小达到p的长度
            if(right - left + 1 == n){
                //判断滑动窗口中的字符的数量是否与cntP相同
                if(Arrays.equals(cntP,windowCnt)){
                    //相同就将起始索引进行记录
                    ans.add(left);
                }
                //不相同就将左窗口后移一位
                windowCnt[s.charAt(left) - 'a']--;
                left++;
            }
        }
        return ans;
    }
}
```
